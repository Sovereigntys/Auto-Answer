# How to Access Auto Answer Mod Config in Mod Menu

## The mod has TWO different interfaces:

### 1. **In-Game Toggle Button** (HUD)
- **Location**: Top-left corner of your screen while playing
- **Function**: Click to enable/disable the auto-answer feature
- **Status**: Shows "Auto Answer: Enabled" or "Auto Answer: Disabled"

### 2. **Mod Menu Config Screen** (Main/Pause Menu)
- **How to Access**:
  1. Press **ESC** to open the pause menu (or go to main menu)
  2. Click the **"Mods"** button
  3. Find **"Auto Answer Mod"** in the mod list (left side)
  4. Click on the mod name to select it
  5. Click the **config/settings icon** (usually a wrench or gear icon) on the right side
  
- **Features Available**:
  - Open Config File
  - Reload Config
  - How to Use (instructions)
  - Answer Delay (adjust timing)

## Troubleshooting

If you don't see "Auto Answer Mod" in the Mod Menu list:
- Make sure `modmenu-13.0.3.jar` is in your mods folder
- Check the game logs for errors
- Verify `autoanswer-1.0.0.jar` is in the mods folder and not disabled
