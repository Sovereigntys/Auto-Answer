üì¶ INSTALLATION:

2. Make sure you have Fabric API installed
3. (Optional) Install Mod Menu for easy config access

üéØ WHAT IT DOES:
- Automatically solves math equations in chat
- Automatically answers your custom prompts

üìù ADDING CUSTOM PROMPTS:

Step 1: Find the config file
  Location: .minecraft/config/autoanswer.json
  (Created automatically on first launch)

Step 2: Edit the file
  {
    "your prompt": "your answer",
    "another prompt": "another answer"
  }

  Example:
  {
    "red": "R3ass0n",
    "blue": "Skywars",
    "green": "MyAnswer"
  }

Step 3: Save and reload
  - Either restart Minecraft
  - Or use Mod Menu ‚Üí Auto Answer Mod ‚Üí Reload Config

üßÆ MATH SOLVING:
  No setup needed! Just works automatically.
  
  Examples:
  - Chat: "5+3" ‚Üí Mod types: "8"
  - Chat: "12*4" ‚Üí Mod types: "48"
  - Chat: "(10+5)*2" ‚Üí Mod types: "30"

üéÆ WITH MOD MENU:
  Press ESC ‚Üí Mods ‚Üí Auto Answer Mod ‚Üí Config
  
  Buttons:
  - Open Config File: Opens the config folder
  - Reload Config: Reloads without restart
  - How to Use: Shows in-game help

‚ö° TIPS:
- Prompts are case-insensitive ("RED" = "red" = "Red")
- Prompts match if contained in message ("what color red?" matches "red")
- There's a 1-second cooldown between auto-answers
- Config file must be valid JSON (use a JSON validator if needed)

‚ùó TROUBLESHOOTING:
- Config not loading? Check JSON syntax at jsonlint.com
- Not answering? Make sure your prompt is in the message
- Still issues? Check logs for "Auto Answer Mod" messages
